 - 1x RP2040

 - 1x EC11 Rotary Encoder
 - 12x 1n4148 diodes
 - 1x USB A to USB C cable
 - 1x soldering iron
 - 11x Black DSA Keycaps
 - 2x Resistor_THT:R_Axial_DIN0204_L3.6mm_D1.6mm_P7.62mm_Horizontal | pull up resistors for oled
 - 1x m2 screw kit on aliexpress
