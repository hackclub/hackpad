### Specifications

# BOM:
- 12x Cherry MX switches
- 1x EC11 Rotary Encoder
- 12x Through-hole 1N4148 Diodes
- 1x 0.91" OLED
- 4x M3x5mx4mm heatset inserts
- 4x M3x16mm screws

Others:
- KMK Firmware
- top - Part1.stl
- bottom - Part1.stl
