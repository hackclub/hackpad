- 4x M3x16 Bolt
- 4x M3 Heatset
- 1x XIAO RP2040
- Custom 3D printed case
- Custom PCB
- 4x MX-Style switches
- 4x Black Blank DSA keycaps
- 2x SK6812 MINI-E LEDs
